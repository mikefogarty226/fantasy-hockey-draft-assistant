# Fantasy Hockey Draft Assistant

A personal dashboard to help with drafting in Yahoo Fantasy Hockey head-to-head categories leagues.

## What it does

This app connects to the Yahoo Fantasy Sports API (read-only) to help make smarter draft-day decisions. It pulls live league data — scoring categories, available players, and draft results — and ranks players based on how well they fit your league's specific category weights.

Planned features:
- Player rankings based on category fit (not just generic overall rankings)
- Configurable category weight sliders, so you can prioritize what matters most to your team
- Roster tracker that flags category gaps as you draft
- "Best available" recommendations updated live as picks happen

## Status

🚧 In development. Currently a personal hobby project for my own Yahoo Fantasy Hockey league, with the possibility of sharing with league mates down the road.

## Tech

- Yahoo Fantasy Sports API (OAuth 2.0, read-only access)
- Fantasy data provided by Yahoo Fantasy
